# 1. WHAT IS SHELLSENSEI?
# ────────────────────────────────────────────────────────────

ShellSensei (殻先生) is an immersive, gamified, interactive web
platform where developers learn, practice, and master CLI
(Command Line Interface) commands — from Linux basics to
advanced cloud tooling — through a deeply themed Japanese
martial arts dojo experience.

It is NOT just a documentation site.
It is NOT just a cheat sheet.
It is NOT just another tutorial platform.

It is a DIGITAL DOJO — a place where users walk in as beginners
(白帯 White Belt) and, through structured training, daily
discipline, real terminal practice, and AI-guided mentorship,
emerge as CLI masters (黒帯 Black Belt).

The platform covers 500+ commands across 14+ CLI ecosystems
including Linux, Bash, Git, Docker, curl, kubectl, Homebrew,
AWS CLI, and more — all unified under one beautifully crafted,
atmospheric, and deeply motivating learning experience.

## The One-Line Pitch
"Duolingo meets Ghost of Tsushima — but for the command line."

## The Elevator Pitch
"ShellSensei is an interactive platform where developers master
CLI commands through a Japanese martial arts dojo experience.
Users progress from White Belt to Black Belt by completing
structured lessons, practicing in a real browser-based terminal,
solving real-world challenges, and getting instant AI-powered
command explanations. It covers everything from basic Linux
commands to Docker, Kubernetes, and cloud CLIs — all wrapped
in a stunning dark-themed UI inspired by Japanese aesthetics
that makes learning the terminal feel like training in a dojo."

## The Theme Rules

The Japanese aesthetic is applied through these principles:

1. AUTHENTICITY over Stereotypes
   ✅ Real Japanese design principles (Wabi-sabi, Ma, Kanso)
   ✅ Actual Japanese terminology (correctly used)
   ✅ Subtle cultural elements (brush strokes, textures)
   ❌ NO cartoon samurai
   ❌ NO ninja clip art
   ❌ NO stereotypical "Asian" fonts
   ❌ NO cultural appropriation

2. SUBTLETY over Heavy-handedness
   ✅ Muted colors with strategic vibrant accents
   ✅ Japanese labels as subtle secondary text
   ✅ Brush-stroke dividers instead of plain lines
   ✅ Texture overlays at very low opacity
   ❌ NOT every element needs a Japanese label
   ❌ NOT in-your-face themed
   ❌ NOT a Japanese restaurant menu

3. ATMOSPHERE over Decoration
   ✅ Dark, focused, atmospheric backgrounds
   ✅ Subtle fog/mist effects
   ✅ Depth through layered gradients
   ✅ The FEELING of being in a dojo
   ❌ NOT cluttered with decorative elements
   ❌ NOT distracting from the learning content

4. FUNCTIONALITY over Aesthetics
   ✅ Theme enhances the learning experience
   ✅ Belt progression motivates continued learning
   ✅ Sensei persona makes AI feel approachable
   ✅ Every themed element serves a purpose
   ❌ NEVER sacrifice usability for theme
   ❌ NEVER make content harder to find because of theme

- 🎯 Duolingo — Gamification, streaks, dopamine loops
- 🏯 Traditional Japanese aesthetics — Wabi-sabi, Ma, Kanso

COLOR PALETTE: "Ink & Ember"
- Deep blacks/dark indigos for backgrounds (like ink/night)
- 竹 Bamboo green for primary actions & terminal text
- 桜 Sakura pink for special elements & terminal prompt
- 炎 Ember orange for streaks & warnings
- 金 Gold for XP/rewards/achievements
- 藍 Indigo for secondary actions
- 朱 Vermillion for danger/errors

TYPOGRAPHY:
- UI: Inter (clean, modern)
- Code/Terminal: JetBrains Mono
- Japanese accents: Noto Serif JP (sparingly)

TEXTURES:
- Subtle noise/grain overlay (washi paper feel)
- Brush-stroke SVG dividers
- Layered gradients for atmospheric depth
- Fog/mist effects on backgrounds

## Key Design Elements

1. Japanese section labels throughout:
   稽古 · TRAINING
   技   · TECHNIQUES

   ── ∿ ── ◯ ── ∿ ──

   「Think of the pipe as a river...」

   sensei@dojo:~/path$
