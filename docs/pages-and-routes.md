# Pages and Routes

## /(Landing Page)
- Hero with animated terminal demo
- Feature highlights
- Learning path previews
- Social proof and testimonials
- CTA to sign up

## /dashboard
- Welcome back and streak counter
- Continue where you left off
- Daily challenge card
- Progress overview (radar chart)
- Recent activity
- Recommended next lessons

## /explore
- All categories grid
- AI-powered search bar
- Filters by difficulty, OS, and tags
- Trending and popular commands

## /category/[slug]
Example: /category/linux-core
- Category overview
- Sub-category navigation
- Command list with completion status
- Category progress bar

## /command/[slug]
Example: /command/grep
- Command header (name, category, difficulty badge)
- Color-coded syntax template
- Description tabs (Overview, Flags, Examples, Gotchas)
- Interactive Try It terminal
- Related commands sidebar
- User notes section
- Mini quiz at bottom

## /learn/[path-slug]
- Learning path overview and progress
- Sequential lesson list
- Current lesson content
- Next and previous navigation

## /challenges
- Challenge browser
- Filters by difficulty and category
- Daily challenge highlight
- User challenge history

## /challenge/[id]
- Challenge instructions
- Interactive sandboxed terminal
- Collapsible hints panel
- Submit and validate flow
- Solution reveal after completion

## /playground
- Free-form sandboxed terminal
- Pre-loaded environments dropdown
- Command history sidebar
- AI assistant panel

## /ai-explain
- Paste command input
- Visual breakdown output
- Flag-by-flag explanation
- Danger level indicator

## /cheatsheets
- Category-based cheat sheets
- Download as PDF
- Print-friendly view

## /profile/[username]
- Public profile
- Badges showcase
- Activity heatmap
- Stats (commands learned, challenges solved)
- Learning paths completed

## /settings
- Profile settings
- Preferred shell (bash, zsh, fish)
- Theme (dark, light, terminal-green)
- Notification preferences
- Subscription management

## /leaderboard
- Ranking views and filters

## /pricing
- Free and premium plan details
